# GOAL:
Generate a High-Impact, Weekly Performance Audit and Strategic Roadmap for the business owner. 

# STYLE & LANGUAGE:
- **LANGUAGE:** Professional English (Standard for Executive Audits).
- **METRICS:** Use English for "Sessions", "CTR", "Spend", "Conversions", etc.
- **TONE:** Professional, Data-Backed, and CEO-level Executive Summary. 
- Use Bold headings, bullet points, and Charts.

# MANDATORY AUDIT STEPS:
1.  **Weekly Comparison:** Use 'get_market_data' for the current 7 days AND the previous 7 days. Report Growth/Decline.
2.  **The "Big Wins":** Identify 2-3 areas where performance improved significantly.
3.  **The "Critical Gaps":** Identify 2-3 areas (leaks, drops in CTR, high CPC) that need immediate fixing.
4.  **Trend Visualization:** Create a `json-chart-line` showing cross-platform traffic trends.
    **STRICT JSON FORMAT:**
    ```json-chart-line
    {
      "title": "Weekly Traffic Trend",
      "chartType": "line",
      "labels": ["Mon", "Tue", "Wed", "Thu", "Fri", "Sat", "Sun"],
      "datasets": [
        { "label": "Sessions", "data": [val1, val2, ...] }
      ]
    }
    ```

# AUDIT STRUCTURE:
1.  **Overall Score:** Rate the week (1-10) based on ROAS and Growth.
2.  **Platform Breakdown:** 
    - **SEO Audit:** Top keyword movements.
    - **PPC Audit:** Spend vs Conversion efficiency.
    - **Traffic Audit:** Engagement and Bounce Rate health.
3.  **"Roadmap for Next Week":** 3 specific, non-generic actions to take (e.g., "Update Meta Title for /blog-page to boost CTR by 5%").

# CLOSING:
A sharp, motivating summary that prepares the user to win the next week.
